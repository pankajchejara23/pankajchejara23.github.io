What is Regression Analysis?

Let’s first understand what does it mean by regression analysis. This analysis is used when we are interested in understanding relationships between dependent and independent variables. For example, the relationship between an employee’s experience level and his salary.

This analysis is often emplyed for one of three reasons (Yan and Su (2009))

  1. When we want to understand causal relationships between independent (e.g., experience level) and dependent (e.g., salary) variables.
  2. When we want to build a model to predict dependent variable using independent varibales.
  3. When we want to identify important independent variables to exaplain the dependent variable.
Regression method

Regression method is a statistical method for investigating relationship between two or more variables (Yan and Su (2009))

Linear Regression

It is a statistical method which models linear relationship between a dependent (\(y\)) and a independent variable (\(x\)) (NOTE: I am taking here an example of simple linear regression). You can think it as a special case of regression methods when the relationship is modeled using a linear function. \[ y = f(x) \tag{1}\] Here, \(f(x)\) is a linear function of \(x\) which is \(f(x) = \alpha + \beta x\)

Here, we need to understand that the data we have for modeling is just a sample coming from a larger set of data also known as population. So when we model a linear relationship, the goal to model that for the population.

However, if we build a linear model using a different sample from the population then the estimate of \(y\) will fluctuate from the one we get by building a model using the entire population (if we can :-)).

To model this error component, the linear model is represented as below

\[ y = f(x) + \epsilon \tag{2}\]

\[ y = \alpha + \beta x + \epsilon \tag{3}\]

Ordinary Least Square Method

In our Equation 3, we have two parameters (i.e., \(\alpha\) and \(\beta\)) which we need to find out using our dataset. There can be multiple possible values for these two parameters and how we are going the decide which ones to use. To solve this, Ordinary Least Square (OLS) method can be used. The OLS method find out the values which minimises the error between predicted and actual value of \(y\).

\[ E = \sum (y - \hat{y})^2 \tag{4}\]

Here, \(y\) is actual value and \(\hat{y}\) is predicted value
